# Cooking Assistant AI Recipe Generator 
A modern web application that helps you create delicious recipes from the ingredients you have on hand. Powered by AI (Ollama) to suggest creative and tasty recipes based on your available ingredients. 

# Tech Stack 
React UI library 
Vite Build tool and dev server
CSS3 Custom Styling 

# AI/ML Integration 
Local LLM alternative Ollama 

##  Prerequisites
Before you begin, ensure you have the following installed:
- Node.js (v18 or higher)
- npm or yarn
- Git
